## 🛠️ Projetos e Exercícios

### 1. 🏧 Simulador de Caixa Eletrônico
Um sistema interativo que simula as operações básicas de um terminal bancário real.

**Funcionalidades:**
* **Verificação de Saldo:** Exibe o saldo atual formatado.
* **Depósito Dinâmico:** Permite adicionar valores ao saldo com validação de entrada positiva.
* **Saque Inteligente:** Realiza saques verificando se há saldo suficiente e se o valor é válido.
* **Interface em Loop:** O programa permanece ativo até que o usuário decida sair.

**Conceitos Praticados:**
* Estruturas de repetição (`while True`).
* Condicionais complexas (`if`, `elif`, `else`).
* Manipulação de tipos de dados (`float`, `int`, `str`).
* Formatação de strings (f-strings com `:.2f`).
